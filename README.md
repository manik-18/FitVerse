# Fitverse

FitVerse is a website that helps people to improve their fitness in an easy and convenient way. It has a big collection of more than 1300 exercises, and you can search for exercises that target specific body parts or muscles. This helps you to create workout plans that suit your personal fitness goals. FitVerse also has a BMI calculator that tells you your Body Mass Index, which helps you to understand your fitness and overall health. If you need help or have questions, there is a chatbot that can assist you right away. The website also has a contact section for any kind of support you may need. FitVerse aims to help people live healthier and more active lives by offering coaching, BMI tracking, chat support, and contact support all in one place.

## Deployment

To access the FitVerse project, simply download it and locate the index.html file. You can find this file by navigating through the following folders:

``` src
    ├── css                     # Contains css files
    ├── img                     # Contains images
    ├── html                    # Contains html files
    │   ├── ...               
    │   ├── index.html          # File to run
    │   └── ...                
    └── js                      # Contains javascript files
```

Make sure to add your own API key and Formspree link for the contact form.
